# 💰 Sales Data Analysis

An end-to-end sales analytics project using SQL, Advanced Excel, and Power BI — performing data cleaning, KPI dashboard development, and trend visualization to support business decision-making.

---

## 📌 Project Overview

This project analyzes sales transaction data to understand:
- Sales performance across products, regions, and time periods
- Key revenue drivers and underperforming segments
- Trends and forecasts to guide quarterly business decisions
- Data quality improvements through cleaning and validation

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| SQL / SQLite | Data extraction and querying |
| Microsoft Excel (Advanced) | Pivot tables, VLOOKUP, INDEX-MATCH, dashboards |
| Power BI | Interactive KPI dashboards and reports |
| Python (Pandas) | Data cleaning and validation |
| Git & GitHub | Version control |

---

## 🔍 Key Analysis Steps

1. **Data Extraction** — Wrote SQL queries to pull and filter sales records by product, region, and date
2. **Data Cleaning & Validation** — Used Python and Excel functions (VLOOKUP, Conditional Formatting) to fix errors and ensure data integrity
3. **Pivot Analysis** — Built pivot tables in Excel to summarize sales by category, region, and time
4. **Dashboard Development** — Integrated Excel with Power BI for interactive KPI tracking
5. **Insights & Forecasting** — Identified trends and produced forecasts for quarterly planning

---

## 📊 Key Insights

- 🏆 Top-performing product categories and regions identified across all quarters
- 📉 Underperforming segments flagged for strategic review
- 📆 Clear **seasonal sales patterns** discovered through time-series analysis
- ✅ Data cleaning improved **accuracy and consistency** across all records

## ▶️ How to Run

```bash
# Clone the repository
git clone https://github.com/madhu250205/Sales_Data_Analysis.git

# Navigate to the project folder
cd Sales_Data_Analysis

# Install required Python libraries
pip install pandas numpy jupyter

# Launch Jupyter Notebook for data cleaning
jupyter notebook
```

> For the Excel dashboard, open `excel/sales_dashboard.xlsx` in Microsoft Excel.  
> For the Power BI report, open the `.pbix` file in Power BI Desktop.
